\frametitle{How is AGI different from LLMs?}
	\begin{itemize}
		\item Reinforcement learning is like a loop.  Its ``knowledge'' changes slowly over many cycles.  Think of it like a washing-machine rinsing out dirt in the clothes.
		\begin{center}
			\includegraphics[scale=0.1]{washing-machine.png}
		\end{center}
		\item Over many cycles, the knowledge in the machine gets more and more correct, removing internal contradictions.
		\item This is how to solve the problem of \textbf{hallucinations}.
		\item LLM: more bulky \qquad AGI: more compact
	\end{itemize}
\end{frame}

\begin{frame}
	\frametitle{What}
	\begin{itemize}
		\item 我翻看了我 2023 年遞交的 HKAI lab 申請，覺得很悲傷。 三年後我更進一步解決了 AGI 問題，但我的付出仍然被忽視。 我覺得我沒有得罪任何人（除了一些相對微小的事），只是我的努力做到了別人做不到的事，而惹起別人的反感。
		\item 很遺憾我的團隊也暫時只有我一人。 我曾經在微信有過 \~160 人的羣組，但後來因爲我在羣內談 民主 和 言論自由 而被永久封號，失去了和他們的聯絡。 在香港我也找不到一個對 AGI 有興趣的人，這包括各大學的很多個教授。 自從 2004 年我從美國回港，這 22 年來我幾乎天天過着以淚洗臉（但並非真的流淚）的日子。 我不知道這地方受了什麼詛咒，以致沒有一個人對 AGI 有興趣。
		\item
	\end{itemize}
\end{frame}

\begin{frame}
	\frametitle{The AGI ethics problem is even harder than AGI}

	\begin{itemize}
		\item 其實關於今次這個項目申請的最大問題是：好多人認爲香港搞 AGI 冇出路，因爲這領域被美國壟斷
		\item 很多香港人亦自我認同某特定的角色，就是吃美國吃剩的「麪包碎」
		\item 當然，知道一個問題的存在，並不表示一定能改變它
		\item 是以很多香港人漸漸潛移默化而形成了種族自卑（很多時不自覺的）
		\item 問題關鍵是：究竟有沒有出路？路徑是否存在？
		\item 而這牽涉的問題是：中國是否必然趨向落後？
		\item 落後的定義是：民衆的思想與「真實」之間的距離。 落後國家的特點是人民思想迷信，浪費時間在低效率的消耗上
		\item 古代文明到了今天衰落，這似乎不是生物界或 DNA 的現象，而是文化上的現象，是存在於人們的思想（教育）裏；能改變思想則不會落後
